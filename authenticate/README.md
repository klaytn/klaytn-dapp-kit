# Sign and Verify a message using web3modal with metamask for Baobab network

Steps to run the application
`cd authenticate`

1. **Setup Application**
    - `yarn install`
    - `yarn start`
    - Open application in browser `http://localhost:3000`
2. Login with wallet connected to Baobab from UI
